# Archigram
Archigram is a simple tool to help with drawing and visualizing diagrams.  Original purpose is to help with drawing software architecture diagrams.

# Installation

All frontend code is in 'client'. Currently using:
- d3
- Webpack
- Babel 


`cd client`
`npm install`
`npm run start`
